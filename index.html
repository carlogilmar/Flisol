GNU Octave

Es un lenguaje de programación para realizar operaciones numéricas, liberada bajo la licencia GNU. Fue diseñado para resolver problemas matemáticos.

Instalación sencilla:

			sudo apt-get install octave
			brew install octave

Graficando

 y = sin (angles)
 plot(angles, y)
plot(angles, y, 'ro'); grid on
plot(angles, y, 'ro--', angles, cos(angles), 'go-'); grid on
legend('sin', 'cos')
